# BGM (trilhas livres de direitos)

Coloque aqui arquivos `.mp3` de trilhas livres de direitos (ex.: Uppbeat, Pixabay Music).
O endpoint `/bgm/tracks` (app/main.py) lista automaticamente os `.mp3` desta pasta.

> Integração real com Epidemic Sound / Uppbeat API está fora do escopo do MVP —
> este diretório é o mock local. Não versione os arquivos de áudio (ver .gitignore).

Exemplo para testar a transform chain:
```
sample_bgm.mp3
```
